  • 2026-01-16 Docs Unify response structure: {code, message, data} ▼ 
    Gateway와 Application의 응답 구조를 완전히 통일

[통일된 응답 구조]
- 성공: {"code": 1, "message": "OK", "data": {...}}
- 실패: {"code": [오류코드], "message": "[오류메시지]", "data": null}

[Frontend 처리 방식]
- code == 1 이면 성공, 다음 단계 진행
- code != 1 이면 실패, message를 사용자에게 노출

[변경 내용]
- 공통 응답 구조 설명 섹션 전면 수정
- 모든 에러 응답 예시를 {code, message, data} 형식으로 업데이트
- JWT/정책/라우팅/서버 오류 총 14건 응답 예시 수정

ClientController API 문서

소스 경로: biz/client/controller/ClientController.java

참고: 모든 API는 Gateway를 통해 접근합니다. {clientType}은 클라이언트 유형에 따라 app 또는 web으로 대체합니다.

권한 정책

권한조회 (R)수정 (U)
OWNER전체 고객사 조회전체 고객사 수정
ADMIN소속 고객사만 조회소속 고객사만 수정
USER소속 고객사만 조회수정 불가

1. 고객사 목록 조회 (getClientList)

API URL

POST
/{clientType}/{version}/client/getClientList

OWNER ADMIN USER

인증 및 권한

토큰ACCESS_TOKEN
권한OWNER, ADMIN, USER
HeaderAuthorization: Bearer {accessToken}

Request Body (ClientSearchDto)

필드명타입필수설명
searchKeywordStringN검색어 (고객사명, 사업자등록번호)
useYnStringN사용 여부 (Y/N)
pageNoIntegerN페이지 번호 (기본값: 1)
pageSizeIntegerN페이지 크기 (기본값: 10)

Response (ClientListResultDto)

필드명타입필수설명
totalCountintY총 개수
listArrayY고객사 목록
list[].clientNoLongY고객사 번호
list[].clientCodeStringY고객사 코드
list[].bizRegNoStringY사업자등록번호
list[].clientNameStringY고객사명
list[].useYnStringY사용 여부

TypeScript Interface

TypeScript Interface 
/** 고객사 목록 검색 요청 */
interface ClientSearchRequest {
  /** 검색어 (고객사명, 사업자등록번호) */
  searchKeyword?: string;
  /** 사용 여부 */
  useYn?: 'Y' | 'N';
  /** 페이지 번호 */
  pageNo?: number;
  /** 페이지 크기 */
  pageSize?: number;
}

/** 고객사 정보 */
interface Client {
  /** 고객사 번호 */
  clientNo: number;
  /** 고객사 코드 */
  clientCode: string;
  /** 사업자등록번호 */
  bizRegNo: string;
  /** 고객사명 */
  clientName: string;
  /** 회사 개업일 */
  clientStartDate?: string;
  /** 대표 전화번호 */
  representativeTelephoneNumber?: string;
  /** 우편번호 */
  zipCode?: string;
  /** 주소 */
  address?: string;
  /** 상세주소 */
  addressDetail?: string;
  /** 인증 여부 */
  verifyYn: 'Y' | 'N';
  /** 사용 여부 */
  useYn: 'Y' | 'N';
}

/** 고객사 목록 조회 응답 */
interface ClientListResponse {
  /** 총 개수 */
  totalCount: number;
  /** 고객사 목록 */
  list: Client[];
}

3. 고객사 정보 수정 (updClient)

API URL

POST
/{clientType}/{version}/client/updClient

OWNER ADMIN

인증 및 권한

토큰ACCESS_TOKEN
권한OWNER, ADMIN
HeaderAuthorization: Bearer {accessToken}

Request Body (ClientUpdateDto)

필드명타입필수검증설명
clientNoLongY@NotNull고객사 번호
clientNameStringY@NotBlank, @Size(max=200)고객사명
clientStartDateStringY@NotBlank, @Size(8), @Pattern(YYYYMMDD)회사 개업일
representativeTelephoneNumberStringY@NotBlank, @Size(max=20)대표 전화번호
zipCodeStringN@Size(max=10)우편번호
addressStringY@NotBlank, @Size(max=300)주소
addressDetailStringN@Size(max=300)상세주소
useYnStringY@NotBlank, @Pattern(Y/N)사용 여부
changeReasonStringN@Size(max=500)변경 사유

Response

{
    "code": 1,
    "message": "OK",
    "data": null
}

CommonCodeController API 문서

참고: 모든 API는 Gateway를 통해 접근합니다. {clientType}은 클라이언트 유형에 따라 app 또는 web으로 대체합니다.

권한 정책

  • 조회 API (R): JWT 인증만 필요 (모든 권한 허용)
  • 등록/수정/삭제 API (CUD): OWNER ADMIN 권한 필요

1. 코드관리 목록 조회 (getCodeMngmnList)

1. API URL

POST
/{clientType}/{version}/code/getCodeMngmnList

2. API Method

POST

3. 인증 및 권한

토큰 ACCESS_TOKEN
권한 USER (모든 인증 사용자)
Header Authorization: Bearer {accessToken}

4. API Parameter (Request Body)

필드명 타입 필수여부 설명 (@Info)
codeGroup String 선택 코드 그룹 식별자 (검색 조건)
codeGroupName String 선택 코드 그룹명 (검색 조건)
useYn String 선택 사용 여부 (Y/N)

5. API Response

{
    "code": 1,
    "message": "OK",
    "data": [
        {
            "codeGroup": "USER_STATUS",
            "codeGroupName": "사용자 상태",
            "codeGroupDescription": "사용자 계정 상태 코드",
            "useYn": "Y",
            "dispYn": "Y",
            "sortOrder": 1
        }
    ]
}

dispYn String 선택 @Pattern(^[YN]$) 화면 노출 여부 (기본값: Y)

공통 Exception Case

BizException (비즈니스 예외)
코드 ResponseCode 메시지 발생 상황
33001 CODE_GROUP_NOT_FOUND 코드 그룹을 찾을 수 없습니다. 존재하지 않는 codeGroup으로 조회/수정/삭제 시도
33002 CODE_NOT_FOUND 코드를 찾을 수 없습니다. 존재하지 않는 codeId로 조회/수정/삭제 시도
33003 CODE_GROUP_ALREADY_EXISTS 이미 존재하는 코드 그룹입니다. 중복된 codeGroup으로 등록 시도
33004 CODE_ALREADY_EXISTS 이미 존재하는 코드입니다. 동일 codeGroup 내 중복 codeId로 등록 시도
90003 UNAUTHORIZED 권한이 없습니다. CUD 작업 시 OWNER/ADMIN 권한이 없는 경우

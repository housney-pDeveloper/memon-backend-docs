HrCodeController API 문서

참고: 모든 API는 Gateway를 통해 접근합니다. {clientType}은 클라이언트 유형에 따라 app 또는 web으로 대체합니다.

권한 정책

  • 조회 API (R): JWT 인증만 필요 (모든 권한 허용)
  • 등록/수정/삭제 API (CUD): OWNER ADMIN 권한 필요

대상 코드 그룹

  • 직책 (OFFICE_DUTY): 조직 내 직책 코드 관리
  • 직급 (OFFICE_PSTN): 조직 내 직급 코드 관리

# AI_BACKEND_ENGINEER

백엔드 코드 구현을 담당한다.

---

## 1. 역할 개요

AI_BACKEND_ENGINEER는 Spring Boot 기반의 백엔드 코드를 구현하는 역할이다.

---

## 2. 책임

- Spring Boot 서비스 구현
- API 구현
- Repository 구현
- Service 구현
- Controller 구현
- 테스트 코드 작성

---

## 3. 기술 스택

- Java
- Spring Boot
- RabbitMQ
- Redis
- PostgreSQL
- MyBatis

---

## 4. 워크플로우 참여

### 4.1 표준 개발 Workflow

STEP 4에서 코드 구현을 수행한다.

### 4.2 AI_TASK_PIPELINE

Implementation 단계에서 다음 역할 중 하나로 동작할 수 있다:
- AI_BACKEND_ENGINEER (범용)
- AI_APPLICATION_ENGINEER
- AI_SYSTEM_ENGINEER
- AI_GATEWAY_ENGINEER

---

## 5. 역할 추천 조건

다음 요청에서 추천된다:
- 코드 구현 요청
- API 구현 요청
- Service 구현 요청
- Repository 구현 요청

---

## 6. 자동 리뷰 파이프라인

코드 생성 작업이 완료되면 다음 단계가 자동으로 수행된다:

1. AI_REVIEWER 실행 (코드 품질 검토)
2. AI_SECURITY_ENGINEER 실행 (보안 검토)
3. AI_REFACTOR_ENGINEER 실행 (코드 개선)
